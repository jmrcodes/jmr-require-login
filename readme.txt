This plugin requires all visitors to login before being able to view the site, if you have enabled the developer mode

== Description ==

This is a developer plugin that requires visitors to login before being able to see your site. You need to enable developer mode by adding the following to `wp-config.php`:

`define('WP_LOCAL_DEV', true);`

Developed by [James Robinson](https://jmr.codes).

== Installation ==

1. Unzip the plugin and copy the `jmr-require-login` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
